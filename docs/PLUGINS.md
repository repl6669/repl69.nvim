## Usage

Enable plugins using their config key:

```lua
require("repl69").setup({
  plugins = {
    -- Auto-detect (recommended)
    auto = true,

    -- Or enable specific plugins
    telescope = true,
    blink = true,
    gitsigns = true,
  }
})
```

For detailed configuration examples, see the [Plugin Guide](PLUGIN_GUIDE.md).
